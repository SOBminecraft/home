Farms and Claims


All farms mentioned below are categorized under Common Farms.
These farms can be built by any player and resources produced by these farms are open for everyone to take. No player can claim these farms for themselves, even if they were built by them.

* - small scale versions

  1. Endermen Farm
    Endermen farm for Ender pearls and/or xp.
    Not to be built on central ender island.
    Only one farm can exist in the world.
    0 Built
  2. Hoglin Farm
    Hoglin farm for porkchops and/or leather.
    Only one farm can exist in the world.
    0 Built
  3. Item Dupers
    Farms that make use of dupping mechanics: Sand/Red Sand/Gravel/Concrete Powder, Rails/Powered Rails, Carpets.
    0 Built
  4. General Mob Farm *
    General mob farm for mobs.
    No restrictions on how many of these can be built.
    0 Built
  5. Cobblestone Farm *
    For cobblestone
    No restrictions on how many of these can be built.
    0 Built
